# `.cursor/plans/`

Scratch space for **working planning artifacts** produced while implementing a task.
Working plans use the `*.plan.md` suffix (the Claude Code mirror is `.claude/plans/`,
which uses plain `*.md`).

These are throwaway, per-task working docs — **not** authoritative documentation and
**not** templates. The authoritative project documents live in `.claude/docs/`. When
authoring a new design doc, adapt the relevant worked doc under `.claude/docs/` rather
than a blank skeleton.
