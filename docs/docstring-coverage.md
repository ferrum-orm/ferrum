# Docstring Coverage Audit — `python/ferrum`

Coverage of **public** symbols (names not prefixed with `_`) across the Ferrum Python
package. Generated by AST inspection over `python/ferrum/**/*.py`.

## Summary

| Category | Documented | Total | Coverage |
|----------|-----------:|------:|---------:|
| Public functions / methods | 46 | 46 | **100%** |
| Public classes | 23 | 23 | **100%** |

Module-level docstrings: every module under `python/ferrum/` has one.

None. The two previously-open gaps were closed:

## Reproduce

Run from the repo root:

```bash
python3 - <<'PY'
import ast, glob
tot_fn=doc_fn=tot_cls=doc_cls=0
missing=[]
for fp in sorted(glob.glob("python/ferrum/**/*.py", recursive=True)):
    tree = ast.parse(open(fp).read(), fp)
    for node in ast.walk(tree):
        if isinstance(node, (ast.FunctionDef, ast.AsyncFunctionDef)):
            if node.name.startswith("_") and node.name not in ("__aenter__",):
                continue
            tot_fn += 1
            if ast.get_docstring(node): doc_fn += 1
            else: missing.append((fp, node.lineno, "func", node.name))
        elif isinstance(node, ast.ClassDef):
            if node.name.startswith("_"):
                continue
            tot_cls += 1
            if ast.get_docstring(node): doc_cls += 1
            else: missing.append((fp, node.lineno, "class", node.name))
print(f"functions/methods: {doc_fn}/{tot_fn} ({100*doc_fn/tot_fn:.1f}%)")
print(f"classes:           {doc_cls}/{tot_cls} ({100*doc_cls/tot_cls:.1f}%)")
for fp, ln, kind, name in missing:
    print(f"  {fp}:{ln}  {kind} {name}")
PY
```

The audit treats most dunder and `_`-prefixed symbols as internal (not counted), except
`__aenter__`, which is surfaced because it is part of the public async-context-manager
protocol. Adjust the filter in the script if you want a stricter or looser definition of
"public".
