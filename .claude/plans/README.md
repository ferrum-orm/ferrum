# `.claude/plans/`

Scratch space for **working planning artifacts** produced while implementing a task
(per `CLAUDE.md`: "Put planning artifacts in `.claude/plans/` as plain `*.md`").

These are throwaway, per-task working docs — **not** authoritative documentation and
**not** templates. The authoritative project documents live in `.claude/docs/`
(architecture, data model, migrations, PRD, etc.). When authoring a new design doc,
adapt the relevant worked doc under `.claude/docs/` rather than a blank skeleton.

The matching Cursor location is `.cursor/plans/` (working plans use the `*.plan.md` suffix).
